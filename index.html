Estrutura Básica do HTML

Html consiste básicamente em três partes começando pela tag <'html'><'/html'> Serve para o navegador entender que se trata de um arquivo html. Assim tudo que tiver entre essa tag é o seu documento html.

Outra parte é abertura e fechamento da tag <'head'><'head'>, que define o cabeçalho do documento. O conteúdo nesse espaço não é visível no browser, mas contém instruções sobre seu conteúdo e comportamento. Dentro dessa tag, por exemplo, podem ser inseridas folhas de estilo e scripts.

A terceira e não menos importante é a tag <'body'><'/body'> que é importante pois ela vem para marcar o espaço no qual deve estar contido o conteúdo visual da página. As demais tags que representam texto, botões etc. devem ser adicionadas nesse intervalo.

Principais TAGS de texto

As tags de texto definem titulos, parágrafos, citações, edição do texto etc. Vamos conhecê-las:

<'h1'> /'h1'> Atribuida para titulos tendo assim variações da mesma tag como se fosse niveis de titulos: Titulo <'h1>, subtitulos<'h2'>, sub e subtitulo<'h3> e assim por diante.

Responsavel por citação ou observasão temos a tag <'bloquote'>.

Temos as de edição do elemento de texto que são:

<'strong>Pra deixar em negrito

<'i>Para italico

<'mark>Para marcar um elemento do texto

E essas são apenas algumas tags para edição de texto do seu html.

Listas Ordenadas e Não Ordenadas

A principal diferença entre a lista ordenada e a lista não-ordenada está na maneira com que os itens de cada lista são apresentados - a lista ordenada enumera os elementos; a lista não-ordenada coloca marcadores antes de cada elemento.

Ordenadas

  1. Uma lista ordenada é delimitada pelas Tags <'OL>

  2. Cada item da lista é delimitado pelas Tags <'LI>.

Não ordenadas

  • A estrutura da lista não-ordenada é semelhante a da lista ordenada

  • porém, ao invés de ser construída com o elemento OL, é construída com a tag UL.
